## 🚀 How to Use This Repository

### Clone the Repository
```bash
git clone https://github.com/DerrickOmwanza/fahamu-shamba.git
cd fahamu-shamba
```

### Install Dependencies
```bash
# For backend
cd backend
npm install

# For frontend
cd frontend
npm install
```

### Run Locally
```bash
# Backend (if applicable)
npm start

# Frontend (if applicable)
npm start
```

### Deploy to Vercel
```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod
```
