🌈 DYNAMIC BACKGROUND SYSTEM

Automatic weather detection → background image selection:

  "Thunderstorm" → stormy.avif + dark overlay
       ↓
  "Rain"         → rainy.avif + dark overlay
       ↓
  "Cloudy"       → cloudy.avif + dark overlay
       ↓
  "Sunny"        → sunny.jpg + dark overlay
       ↓
  "Snow"         → snowy.avif + dark overlay
       ↓
  "Fog"          → foggy.avif + dark overlay

• Smooth 0.5s transition between weather types
• Dark overlay (rgba 0,0,0,0.3) for text readability
• Fallback colors if images don't load
• Works on all devices (desktop, tablet, mobile)
