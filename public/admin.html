Security posture

Session-only storage Access, refresh, CSRF, and session identifiers remain in sessionStorage, not persistent local storage.
Bound session checks Authenticated admin routes require matching session ID and request fingerprint validation on the server.
Mutation hardening Alert creation, admin creation, and logout submit CSRF headers on every state-changing request.
